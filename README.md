# Best-Professor-Review-Tools

这几年看身边太多人选导师踩坑了，学术圈信息太闭塞，选错导师，研究生真的比高三还难熬。整理这个列表，希望能帮大家在做决定前有个查口碑的地方。

## 1. 国内导师评价

在国内读研，选导师基本上就是选老板。除了要避开人品差的，能够找到真正的学术大牛也是关键。

**导师评价网**
* 网站：http://daoshipingjia.net/

  这个网站应该是目前国内最好用的导师评价查询工具。

1. **导师搜索**
   网站对导师有多个核心评分维度：
   
   * 学术水平：是不是真大牛，还是只会在外面吹水。
   * 科研经费：没钱的组真的很难做实验。
   * 师生关系：最重要的一项！看是不是Push怪，会不会把学生当苦力用。
   * 学生前途：毕业生的去向好不好，允不允许出去实习。
   
   看评论区比看分数更重要

3. **学术大牛排行榜**
   * 地址：https://daoshipingjia.net/scholars
   
   汇集了全球 144 个领域的学术大牛，以前找导师得去翻顶会论文、效率很低，这个排行榜把数据透明化了：
   
   * 查学校实力：可以按学校看大牛数量。
   * 查导师画像：每个入选的学者都有详细的五维画像（排名、h指数、独立性、复合得分、引用）。
     
   这个功能对快速锁定真正有实力的科研大佬用处比较大。

## 2. 国外教授评价 (针对留学/选课)

出国留学的同学，尤其是读本科和授课型硕士的。

**RateMyProfessors**
* 网站：https://www.ratemyprofessors.com/

全球最大的教授评分库，基本涵盖了北美和英国的主流学校。
选课前进去搜一下教授名字，主要看以下三个指标：

1. Quality：1-5分，分数太低的直接避雷。
2. Difficulty：1-5分。注意有时候难度高是因为学得到东西，要结合Quality一起看。
3. Would Take Again：这个百分比最能说明问题。

看一眼下面的Tags，比如是否有 "Textbook Usage" (是否必须买书)、"Mandatory Attendance" (强制考勤) 

主要反映上课体验，对找科研型的PhD导师参考价值有限。

## 3. 其他隐形渠道 (补充)

有些导师评价也可以在论坛里查。

* **一亩三分地**
  留学圈特别是CS/EE专业的必看。去院系介绍版块搜学校或导师名字。如果有坑，通常会有警告帖。

* **知乎**
  直接在搜索框搜导师名字或者某某学校某某课题组体验。重点看那些匿名的回答。
